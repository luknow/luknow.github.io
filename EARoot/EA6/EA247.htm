The user has the ability to report:<br/><ul>
<li>Chat,</li><li>Profile.</li></ul>
The user has the ability to block, unblock profiles. <br/>The user has the ability to block chat (which is equivalent to removing the matched profile).<br/>The user has the ability to add to favorites list profile that is in his friend circle.<br/>The user has the ability to add to favorites list profile that is in his Instagram's followers circle.<br/>The user has the ability to remove from favorites list mentioned above profiles.<br/>The user has the ability to modify viewing candidate profiles which are friends of a friend to remain anonymity.<br/>
