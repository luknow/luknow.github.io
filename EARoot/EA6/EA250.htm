The user has the ability to view the profile in two ways:<br/><ul>
<li>Full profile version,</li></ul>
<ul>
<li>Shortened profile version.</li></ul>
For the following profiles:<br/><ul>
<li>Candidate profile,</li></ul>
<ul>
<li>Profile of a candidate who sent a like,</li></ul>
<ul>
<li>Matched profile.</li></ul>
Additionally, the user has the option to view the profile of another user who belongs to the same Facebook group as the user or who participated in the same Facebook event as the user.<br/>

Scenario Type
1. Viewing the profile that sent a like - shortened version Basic Path
Details:
<ol>
<li>The user navigates to profiles that sent a like section.</li><li>Shortened version of the profile that sent a like is presented.</li></ol><p/>
1.1. Viewing the profile that sent a like - full version Alternate
Details:
<ol>
<li>The user navigates to profiles that sent a like section.</li><li>Shortened version of the profile that sent a like is presented.</li><li>The user decides to view full profile version.</li><li>Full version of the profile that sent a like is presented.</li></ol><p/>
1.1.1. Viewing the profile that sent a like - full version with photo comment Alternate
Details:
<ol>
<li>Full version with photo comment of the profile that sent a like is presented.</li></ol><p/>
2. Viewing candidate profile - shortened version Basic Path
Details:
<ol>
<li>The user navigates to candidate profiles section.</li><li>Shortened version of the candidate profile is presented.</li></ol><p/>
2.1. Viewing the candidate profile - full version Alternate
Details:
<ol>
<li>The user navigates to candidate profiles section.</li><li>Shortened version of the candidate profile is presented.</li><li>The user decides to view full profile version.</li><li>Full version of the candidate profile is presented.</li></ol><p/>
2.2. Viewing the candidate profile - criterion is not met Exception
Details:
<ol>
<li>The user navigates to candidate profiles section.</li><li>The user receives a message that: application is unable to search for profiles based on the provided criteria.</li><li>The user has been asked to manage profile preferences.</li></ol><p/>
3. Viewing matched profile - full version Basic Path
Details:
<ol>
<li>The user navigates to matched profiles section.</li><li>The user decides to view full profile version.</li><li>Full version of the matched profile is presented.</li></ol><p/>
4. Viewing profiles in the same Facebook group Basic Path
Details:
<ol>
<li>The user navigates to profiles in the same Facebook group section.</li><li>The user selects from which group the profiles should be searched.</li><li>Shortened version of the candidates profiles that are in the same Facebook group is presented.</li></ol><p/>
5. Viewing profiles in the same Facebook event Basic Path
Details:
<ol>
<li>The user navigates to profiles in the same Facebook event section.</li><li>The user selects from which event the profiles should be searched.</li><li>Shortened version of the candidates profiles that participated in the same Facebook event is presented.</li></ol><p/>
