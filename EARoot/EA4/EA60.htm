Facebook dating is an application available within the mobile version of Facebook.<br/>The goal of the application is to help people connect, including those with similar interests and views.<br/><br/>What sets it apart from other existing applications of this kind (especially in the realm of dating applications and its main competitor, Tinder, which currently holds a monopoly) is that:<br/><ul>
<li>It is completely free (with no hidden microtransactions),</li></ul>
<ul>
<li>Likes appear instantly (no need to swipe through profiles),</li></ul>
<ul>
<li>Profiles contain more information (users are not reduced to just photos).</li></ul>
<br/>The application was introduced in Poland in October 2020.<br/><br/>This document serves as a proposal for the applications documentation, showcasing my approach to analysis and modeling.<br/>
