The user has the option to vote in four ways:<br/><ul>
<li>Positive vote,</li></ul>
<ul>
<li>Negative vote,</li></ul>
<ul>
<li>Comment vote,</li></ul>
<ul>
<li>Super vote.</li></ul>
The user has the option to vote again for a profile that was rejected in the first vote. In case of a second rejection, that profile will no longer be displayed to the user.<br/><br/>

Constraint Type Status
Pre condition Pre-condition Approved
Details:
The user must be logged into the application.<br/>
Vote limitations Invariant Approved
Details:
If the user exceeds the daily limit of 50 positive votes or comment votes combined, they will not be able to submit any further votes for the next 24 hours.<br/>
Super vote limitations Invariant Approved
Details:
If the user exceeds the daily limit of 3 super votes, they will not be able to submit any further votes for the next 24 hours.<br/><br/><br/>
Comment vote limitations Invariant Approved
Details:
The user can vote with a comment only on a candidate's profile in the full profile version or on a candidate's profile who sent a like in the full profile version.<br/><br/>
Comment vote limitations - character limitations Invariant Approved
Details:
The user can enter a comment of up to 200 characters. If this limit is exceeded, a message will appear informing the user that the limit has been exceeded.<br/>
Post condition Post-condition Approved
Details:
Information about the positive vote is saved in the database between the profile of the user sending the like and the profile of the user receiving the like.<br/>
